## Object-Oriented Programming (OOP) using C++.

Instead of only studying concepts theoretically, I created a dedicated repository where I implemented and practiced each topic through code examples and mini design exercises.

### Topics covered:
* Classes & Objects
* Constructors and Constructor Overloading
* Encapsulation
* Static Members
* Inheritance
* Polymorphism
* Abstraction
* Interfaces
* Aggregation and Composition

### Implemented and practiced these concepts through real examples:
* Payment system design with interchangeable payment strategies
* Shape/Animal hierarchies using abstraction and polymorphism
* Composition vs inheritance use cases
* Constructor and access-control examples

### A few concepts that especially improved my understanding were:
* Runtime polymorphism and dynamic binding
* Upcasting with base-class pointers
* When to use inheritance vs composition
* Designing systems using interfaces and abstraction

The repository now serves as my personal OOP reference and revision guide.

Next, I'll be shifting my focus towards applying these OOP concepts in larger software and machine learning projects.

Learning by building has been far more effective than learning by reading alone.
